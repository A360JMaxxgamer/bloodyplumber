In diesem Paket ist der Leveleditor, das Spiel sowie die Doku enthalten. Wir haben zusätzlich das Framework Monogame hinzugefügt, da man eine finale exe nur über
den Windows Store erstellen und veröffentlichen kann.
Dieses müssen Sie installiert haben um das Spiel starten zu können. Des Weiteren müssen Sie eventuell XNA bzw. das .Net Framework installieren.
Zudem wird Sie Visual Studio vor dem Ausführen dazu auffordern eine Entwicklerlizenz zu erstellen, sofern Sie diese nicht schon haben.
Wir hoffen das Sie das Spiel so starten können andernfalls, können Sie auch ohne weiteres Kontakt mit uns aufnehmen und das Spiel auf einem unserer Geräte testen!
